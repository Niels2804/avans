## Robot information

### Sensors
- Front-center Ultrasonic: `D24`
- Front-right Ultrasonic `D26`
- Front-left Ultrasonic `D22`
- Back-center Ultrasonic `D18`
- Motion Sensor `D16`
- Red LED Button `D5`
- Speaker `PWM`
- GyroCompass (Arduino IMU 9DOF) `I2C` midden `0x68`
- Display Grove-16x2 LCD (Black on Red) `I2C` links `0x3E`
